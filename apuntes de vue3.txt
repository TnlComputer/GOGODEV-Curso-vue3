PREPROCESADORES

Para instalar el pre procesador  seria

npm i -d  sass-loader sass
npm i -d  less-loader less
npm i -d  styles-loader styles


dependiendo el pre procesador que se usara.

esto si ya tenemos el proyecto, si aun no lo creamos al preyecto, lo elegimos en la instalacion del proyecto.


en el style del componente que nos <button type="button" class="btn btn-secondary" data-bs-container="body" data-bs-toggle="popover" data-bs-placement="top" title="Popup title" data-bs-content="Popup content">

<style scoped>

agregamos
lang='scss' y quedaria

<style scoped lang='scss'>


configurar en el vue.config.js para integrar el scss par anuestro proyecto

quedaria algo asiconst { defineConfig } = require('@vue/cli-service');
module.exports = defineConfig({
  transpileDependencies: true,
  css: {
    loaderOptions: {
      sass: {
        additionalData: `
        @import "@/scss/_variables.scss";
        `
      },
    },
  },
});
